## Installation instructions:

### Linux/Mac:  
pip install theano  
pip install tensorflow  
pip install keras  
conda update --all  

### Windows:
conda create -n tensorflow python=3.5 anaconda  
activate tensorflow  
conda install theano  
conda install mingw libpython  
pip install tensorflow  
pip install keras  
conda update --all
